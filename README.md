# PRODIGY_WD_02
## Task Description
This project is a Stopwatch Web Application created as part of the Prodigy InfoTech Web Development Internship.

## Features
- Start, Pause, and Reset stopwatch
- Lap time tracking
- Responsive UI
- Implemented a dark mode toggle using JavaScript by dynamically applying CSS classes to enhance user experience.

## Technologies Used
- HTML
- CSS
- JavaScript
